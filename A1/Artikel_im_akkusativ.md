<h3>Artikel im akkusativ</h3>

<h4>unbestimmte artikel im akkusativ</h4>

```php

const maskulin = {
  nominativ: 'ein',
  akkusativ: 'einen'
};

const feminin = {
  nominativ: 'eine',
  akkusativ: 'eine'
};

const neutral = {
  nominativ: 'ein',
  akkusativ: 'ein',
};

```


<h4>bestimmte artikel im akkusativ</h4>

```python
Maskulin = {
	"Nominativ": "der",

}

```
