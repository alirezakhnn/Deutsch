<h4>The Plural Article<h4>

```bash

# Example => Die Bucher sind bunt

# bestimmte Artikel im Nominativ

1.maskulin => der(singular) && die(plural)
2.feminin => die(singular) && die(plural)
3.neutrum => das(singular) && die(plural)

```

<h5>Examples</h5>

```python
#Singular
maskulin = f"{Der} Mann"
feminin = f"{die} Frau"
neutrum = f"{die} Baby"

#plural
maskulin = f"{Die} Männer" # r
feminin = f"{Die} Frauen" # e
neutrum = f"{Die} Babys" # s

```
