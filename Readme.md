
```bash
Subjeckt(Nominativ) => 1.Nomen - 2.Pronomen(pronoun)

Nomen => 1.Person  - 2.Sache(thing)

i.e of Person => "Wer" ist das? (Who is this?) - Das ist Peter
i.e of Sache(thing) => "Was" ist das?(what is this?) - Das ist ein Bus.

```

<h4>The Plural Article<h4>

```bash

# Example => Die Bucher sind bunt

# bestimmte Artikel im Nominativ

1.maskulin => der(singular) && die(plural)
2.feminin => die(singular) && die(plural)
3.neutrum => das(singular) && die(plural)

```

<h5>Examples</h5>

```python
#Singular
maskulin = f"{Der} Mann"
feminin = f"{die} Frau"
neutrum = f"{die} Baby"

#plural
maskulin = f"{Die} Männer" # r
feminin = f"{Die} Frauen" # e
neutrum = f"{Die} Babys" # s

```
